# المساهمة في لعبة ألغاز الكلمات العربية

شكرًا لاهتمامك بالمساهمة! 🎉

## قواعد السلوك

هذا المشروع يلتزم بميثاق سلوك المساهمين. بالمشاركة، أنت توافق على الالتزام بهذه القواعد.

## كيف يمكنني المساهمة؟

### الإبلاغ عن الأخطاء
- تأكد من أن الخطأ لم يتم الإبلاغ عنه مسبقًا
- استخدم قالب تقرير الخطأ
- أضف خطوات إعادة الإنتاج

### اقتراح ميزات جديدة
- افتح issue أولاً لمناقشة الميزة
- اشرح الفائدة والقيمة المضافة

### إضافة مستويات
- اتبع تنسيق JSON المحدد
- تأكد من صحة الكلمات باستخدام القاموس
- اختبر المستوى قبل الإرسال

### المساهمة بالكود
1. Fork المشروع
2. أنشئ فرع للميزة (`git checkout -b feature/AmazingFeature`)
3. Commit التغييرات (`git commit -m 'feat: add some AmazingFeature'`)
4. Push إلى الفرع (`git push origin feature/AmazingFeature`)
5. افتح Pull Request

## معايير الكود

- اتبع معايير [Effective Dart](https://dart.dev/effective-dart)
- استخدم `flutter analyze` قبل الإرسال
- اكتب اختبارات للميزات الجديدة
- اتبع مبادئ Clean Architecture
- وثّق الكود العام

## Commit Messages

نستخدم [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` ميزة جديدة
- `fix:` إصلاح خطأ
- `docs:` تغييرات التوثيق فقط
- `style:` تنسيق الكود
- `refactor:` إعادة هيكلة الكود
- `test:` إضافة اختبارات
- `chore:` مهام الصيانة

## الترخيص

بالمساهمة، توافق على أن مساهماتك ستكون مرخصة تحت نفس ترخيص المشروع.
